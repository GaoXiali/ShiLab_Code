ShiLab_Code | 实验室代码托管与数据管理平台

欢迎使用 ShiLab 代码仓库！本平台旨在通过 Git 实现代码同步，减少“重复造轮子”的工作，提升实验室整体科研效率。

📢 重要提示
获取数据：由于 GitHub 不建议存储大型数据集，代码配套的相关数据请联系 gxl 线下索取。

开发说明：目前代码更新频繁，建议各位同学在开始工作前先执行 git pull 获取最新版本。

📁 文件导航
核心开发脚本 (根目录)
    SphericaRecon.m：

        用途：核心开发版代码。

专项文件夹
📂 Spherical Array System：

    说明：针对特定日期采集数据调整好的代码存档。
    优势：下载后可直接处理对应数据，无需手动调参。
    适用对象：主要面向 xzb 等需要快速获取训练数据进行神经网络研究的同学。

📂 Code Source：

    说明：代码开发过程中的辅助函数库及独立功能代码。
    注意：运行 SphericaRecon.m 若提示缺少函数，请在此文件夹中查找。

🚀 快速上手
如果你是第一次使用，请参考以下流程：

    克隆仓库：git clone https://github.com/GaoXiali/ShiLab_Code.git
    获取数据：联系 gxl 拷贝必要的数据集。
    运行环境：确保 MATLAB 已添加 Code Source 文件夹到路径（Set Path）。
